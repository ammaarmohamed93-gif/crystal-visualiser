# 💎 Crystal Structure Visualizer + ✨ AI Professor

A high-fidelity 3D interactive tool designed for **Mechanical Engineering** students to visualize metallic crystal lattices and study for Material Science CAT exams.

## 🛠️ Features
* **Interactive 3D Models:** Toggle between **Simple Cubic (SC)**, **Body-Centered (BCC)**, and **Face-Centered (FCC)** unit cells.
* **Space-Filling Mode:** Visualize the "Hard Sphere" model to understand **Atomic Packing Factor (APF)** and atom-touching geometry.
* **✨ AI Study Assistant:** A built-in "Professor" powered by **Gemini 1.5 Flash** to answer technical questions about coordination numbers, density calculations, and Miller indices.
* **Mathematical Rendering:** All formulas are rendered using **KaTeX** for textbook-quality clarity ($a = 2R$, $\sqrt{3}a = 4R$, etc.).

## 📖 How to Use
1.  **Open the App:** Navigate to the live link or open `index.html` in your browser.
2.  **Enter API Key:** To enable the AI Professor, provide a **Gemini API Key** from [Google AI Studio](https://aistudio.google.com/).
3.  **Explore:** Use your mouse to rotate and zoom into the 3D unit cells.
4.  **Study:** Ask the AI questions like *"How do I calculate the theoretical density of BCC Iron?"* or *"Explain the APF of 0.74 for FCC."*

## 🏗️ Technical Stack
* **Three.js:** For the 3D WebGL rendering engine.
* **Tailwind CSS:** For the modern, responsive UI.
* **Google Gemini API:** For the generative AI educational layer.
* **KaTeX:** For high-speed LaTeX math rendering.
