# Crystal Structure Visualizer & AI Professor 🛑🧊

A lightweight, interactive 3D web application designed to help visualize fundamental crystallographic structures and calculate planar densities. It features a built-in AI assistant, "Prof. Lattice," to explain material science concepts on the fly.

## ✨ Features

* **Interactive 3D Lattices:** Visualize Simple Cubic (SC), Body-Centered Cubic (BCC), Face-Centered Cubic (FCC), and Hexagonal Close-Packed (HCP) unit cells.
* **Space-Filling Mode:** Toggle between standard point-lattice views and accurate space-filling models to visualize Atomic Packing Factor (APF) geometry.
* **Planar Density Calculator:** Input Miller indices (hkl) to dynamically calculate atomic planar densities for specific crystal planes.
* **"Prof. Lattice" AI Assistant:** An integrated chat panel powered by the Google Gemini API. Ask complex material science questions, get derivations, and view mathematical formulas rendered cleanly in the browser.

## 🛠️ Technologies Used

* **Three.js:** For 3D WebGL rendering and orbit controls.
* **Tailwind CSS:** For the sleek, responsive UI overlay.
* **Google Gemini API:** Powers the AI chat functionality.
* **KaTeX & Marked.js:** For parsing Markdown and rendering LaTeX mathematical formulas (like $a = 2R\sqrt{2}$) generated by the AI.

## 🚀 How to Run

This project is built as a single-file web application. There are no build steps, package managers, or backend servers required.

1. Clone or download this repository.
2. Open `crystal visualisr.html` directly in any modern web browser.
3. Use your mouse to rotate, zoom, and pan around the 3D unit cell.

## 🤖 Setting Up the AI Professor

To use the chat feature, you will need a free Google Gemini API key:

1. Get an API key from [Google AI Studio](https://aistudio.google.com/).
2. Open the visualizer in your browser.
3. Click the **Set API Key** button in the top right corner of the chat panel.
4. Paste your key. It will be stored securely in your browser's local storage (it is never sent anywhere except directly to Google's API).
